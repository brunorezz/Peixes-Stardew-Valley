# 🎣 Stardew Valley Fish Guide

Uma aplicação interativa desenvolvida para ajudar os fazendeiros de **Stardew Valley** a catalogar e encontrar peixes de forma eficiente. Chega de perder o horário do *Lúcio* ou esquecer em qual estação o *Peixe-Gato* aparece!

---

## 🚀 Funcionalidades

O guia permite filtrar a vasta fauna aquática da Vila Pelicano através de três eixos principais:

* **Estação:** Filtre por Primavera, Verão, Outono ou Inverno.
* **Localização:** Saiba exatamente onde lançar a linha (Rio, Lago da Montanha, Oceano, etc.).
* **Dificuldade:** Visualize o nível de dificuldade de captura e o padrão de movimento do peixe.
* **Informações Extras:** Exibição de horários, clima necessário e se o peixe é usado no **Conjunto do Tanque de Peixes** (Centro Comunitário).

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando:

* **Frontend:** [React.js](https://reactjs.org/) (ou a tecnologia que usaste)
* **Estilização:** Tailwind CSS / Styled Components
* **Dados:** JSON customizado baseado na Wiki Oficial de Stardew Valley.
* **Ícones:** Lucide React / FontAwesome
